# Kaur-geous-Makeovers

## Project Description

Kaur-geous Makeovers is a portfolio website for a professional beauty studio in Nabha, offering services like bridal makeup, party makeup, nail art, and hair styling.

## Deployment

This project is configured to deploy automatically to **GitHub Pages** using GitHub Actions.

### How to Enable Deployment

1. Go to the **Settings** tab of this repository on GitHub.
2. In the left sidebar, click on **Pages**.
3. Under **Build and deployment** > **Source**, select **GitHub Actions** from the dropdown menu.
4. The deployment will trigger automatically on the next push to the `main` or `master` branch.
   - You can also manually trigger it by going to the **Actions** tab, selecting "Deploy to GitHub Pages", and clicking "Run workflow".

### Local Development

To run this project locally:

1. Clone the repository.
2. Open `index.html` in your browser.
